# ✨ NextWord AI - LSTM Next Word Predictor

A modern, attractive, and intelligent **Next Word Prediction** web application built with **Streamlit** and **LSTM**.

<img width="1918" height="882" alt="Screenshot 2026-05-04 111508" src="https://github.com/user-attachments/assets/09a02c6a-33e1-499e-905c-7ca12b16132a" />


## 🌟 Features

- **Modern & Clean UI** with gradient design
- **Real-time Next Word Suggestions**
- **Adjustable Creativity** (Temperature Control)
- **Click-to-Copy** full sentence
- **Fast & Responsive** interface
- **Top-N Predictions** with confidence scores


## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Model**: LSTM (TensorFlow/Keras)
- **Backend**: Python
- **Preprocessing**: Tokenizer + Padding

## 📁 Project Structure
